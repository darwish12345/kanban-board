# 📋 Kanban Board Pro - Task Management Tool

A Trello-style kanban board application for managing tasks and projects. Create columns, add tasks, drag and drop between columns, and track your progress.

## 📖 About The Project

Kanban Board Pro is a visual task management tool that helps users organize work using the kanban methodology. Create custom columns (To Do, In Progress, Review, Done), add tasks with priorities and labels, and drag them between columns as work progresses.

## ✨ Features

- 🎯 **Drag & Drop** - Move tasks between columns with drag and drop
- 📊 **Multiple Columns** - Create, rename, or delete columns
- ✅ **Task Management** - Add, edit, and delete tasks
- 🏷️ **Labels & Categories** - Assign labels (Bug, Feature, Enhancement, Docs, Design)
- 🔴 **Priority Levels** - Set task priority (Low, Medium, High)
- 📅 **Due Dates** - Set deadlines for tasks
- 💾 **Persistent Storage** - Board saves automatically in browser
- 📤 **Export/Import** - Save board as JSON or load existing boards
- 🎨 **Custom Colors** - Choose colors for each column
- 📱 **Responsive Design** - Works on desktop and tablet

## 🚀 How To Use

### 1. Run the Application
- Open `index.html` in your web browser
- Or use Live Server in VS Code

### 2. Create Your Board
- **Add Column** - Click "Add Column" button
- **Add Task** - Click "+ Add Task" on any column
- **Edit Task** - Click edit icon on any task
- **Delete Task** - Click delete icon on any task

### 3. Manage Tasks
- **Set Priority** - Choose Low, Medium, or High
- **Add Labels** - Select from Bug, Feature, Enhancement, Docs, Design
- **Set Due Date** - Pick deadline for each task
- **Drag Tasks** - Move tasks between columns

### 4. Save & Export
- **Auto-save** - Board saves automatically to browser
- **Export** - Click "Export" to save board as JSON
- **Import** - Click "Import" to load saved boards
- **Reset** - Use "Reset Demo Board" to load sample data
